2. Propuesta de Diseño: Sobrio y Moderno

Un diseño "sobrio" huye de los colores chillones y se basa en la jerarquía visual y el espacio en blanco (o aire).

Paleta de Colores (Estilo "Dark Elegant"):Fondo (Body): #121212 (Negro mate profundo).

Evita el negro puro #000 para que no canse la vista.Superficies (Cards/Nav): #1E1E1E (Gris muy oscuro).

Crea capas visuales.Acento (Botones/Links): #00E676 (Verde Neón Energético) o #2979FF (Azul Eléctrico). Solo un color para resaltar acciones.Texto: #E0E0E0 (Blanco roto) para el cuerpo y #FFFFFF para títulos.Tipografía:Títulos (H1, H2): Montserrat o Oswald (estilo negrita/bold). Transmiten fuerza y estabilidad, ideal para gimnasios.Cuerpo: Inter o Roboto. Son fuentes diseñadas para pantallas que facilitan la lectura de descripciones largas.Tips de Diseño Profesional:Bordes redondeados suaves: Usa rounded-3 o rounded-4 de Bootstrap en lugar de esquinas rectas para un look más moderno.Sombras sutiles: Aunque el fondo sea oscuro, una sombra leve (shadow-sm) en las cards ayuda a dar profundidad.
